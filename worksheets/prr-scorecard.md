# Production Readiness Review Scorecard

Copy this scorecard for each PRR. Enter a score from 0 to 5 and multiply by the weight to calculate a weighted total.

## Scorecard

| Category | Weight | Score (0-5) | Evidence | Notes |
|---|---|---|---|---|
| Service level objectives | 15% | | | |
| Rollback and deployment safety | 15% | | | |
| Feature stores | 15% | | | |
| Model registry | 10% | | | |
| Observability | 20% | | | |
| Cost and efficiency | 10% | | | |
| Runbooks and documentation | 15% | | | |
| **Total** | **100%** | | | |

## Scoring guide

| Score | Meaning |
|---|---|
| 5 | Exceeds requirements; automated, tested, and documented |
| 4 | Meets requirements with minor, tracked gaps |
| 3 | Meets most requirements; some manual steps remain |
| 2 | Significant gaps; missing owner, test, or documentation |
| 1 | Critical gaps; high residual risk |
| 0 | Not started or unknown |
